2. Про голос и распознавание речи (важно)

Dashman — голосовой ассистент.
Когда ты говоришь команды, используется системный сервис
распознавания речи Android.

Проще говоря:
распознавание выполняется через сервис Google SpeechRecognizer;
в этот момент аудио может передаваться сервису Google
для преобразования речи в текст;
Dashman не записывает, не хранит и не анализирует твой голос.

Я не контролирую, что Google делает с аудио — это их сервис и их правила.
Если тебя это не устраивает — просто не используй голосовой ввод.
Никаких скрытых режимов тут нет.

Это отдельный механизм, никак не связанный с Google.

5. Device ID и лицензия

Dashman использует Device ID. Спокойно, без истерик.

Это:
не имя;
не номер телефона;
не доступ к фото, контактам или переписке;
не GPS и не слежка.

Device ID нужен только для того, чтобы:
привязать лицензию к одному устройству;
защитить приложение от пиратства;
чтобы каждый второй умник не раздавал Dashman по чатам.

При первом запуске Device ID отправляется на сервер автора
вместе с запросом на активацию.
После одобрения Dashman проверяет статус лицензии
при каждом последующем запуске приложения —
это занимает секунду и работает в фоне.
Ты этого даже не заметишь.

Никакой аналитики, профилей и "мы знаем, кто ты".
Только Device ID и статус лицензии. Больше ничего.

6. Диагностические логи

Dashman ведёт технический журнал работы (лог).
Он хранится только на твоём устройстве.

Что там есть: технические события — когда запустился,
что выполнил, где споткнулся.
Чего там нет: личные данные, текст напоминаний в полном виде,
голос, контакты.

Лог автоматически никуда не уходит. Вообще никогда.

Он передаётся мне только если ты сам нажмёшь "Связь с автором"
и прикрепишь его к сообщению. Добровольно. Осознанно. По своей воле.

Я использую его только для диагностики и исправления ошибок.
Больше ни для чего — мне буквально неинтересно читать
технические логи просто так.

Параноишь по поводу логов — не прикрепляй. Всё.

7. Интернет — зачем он вообще нужен

Dashman использует интернет:
для системного распознавания речи (см. пункт 2);
для активации и проверки лицензии при каждом запуске;
для функции "Связь с автором" — только когда ты сам пишешь.

Интернет не используется для скрытого сбора данных.
Точка.
